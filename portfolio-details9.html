The Story

The tax offices faced a common challenge: manual scheduling led to inefficiency, long visitor queues, and poor time management. My goal was to build a web application that would allow users to check the real-time availability of officials and schedule appointments accordingly.

Collaborating with a team of three, I led the design and implementation process using the MERN stack (MongoDB, Express.js, React, Node.js). The key feature was an integrated scheduling system that automatically adjusts available time slots based on real-time data.

Key Features

  • Real-time appointment scheduling using MongoDB and React
  • Automated time slot adjustments based on system updates
  • Scalable Dockerized backend to handle multiple office locations

Challenges & Solutions

The major challenge was managing a large number of real-time queries without overloading the database. By implementing efficient indexing and optimizing queries in MongoDB, I reduced database load by 30%, ensuring a smooth user experience even during peak hours.

Impact & Results

By automating the scheduling process, VisitGST reduced visitor wait times by 30% and improved scheduling efficiency by 21%. The project is currently used across multiple tax offices, with plans to scale further.
